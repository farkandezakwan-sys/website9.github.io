## 🚀 Deployment (GitHub Pages)

1. Create a repository (recommended name: `portfolio`)

2. Upload:

   * `index.html`
   * `assets/` folder
   * `README.md`

3. Go to:
   **Settings → Pages**

4. Set:

   * Source → `Deploy from a branch`
   * Branch → `main`
   * Folder → `/ (root)`

5. Your site will be live at:

   ```
   https://yourusername.github.io/portfolio/
   ```
